핵심 UX 피드백 포인트

1. 음성 메모 → 리포트 전환 UX
프로가 FAB 버튼을 눌러 음성 녹음 → 자동 전사 → 리포트 초안 생성까지의 흐름이 직관적인가?
2. 인과 그래프 시각화 이해도
회원이 스윙 요소 간 인과관계 그래프를 쉽게 이해할 수 있는가? 라벨/컬러 직관성은?
3. 신뢰도 3-Tier 표시 방식
T1(High)/T2(Mid)/T3(Low) 측정 신뢰도를 사용자에게 어떻게 보여줄 것인가?
4. 오프라인 → 온라인 전환 UX
오프라인에서 작성한 데이터가 온라인 복귀 시 자연스럽게 동기화되는가? 충돌 처리는?
5. 프로-회원 연결 온보딩
프로가 회원을 초대하는 플로우, 회원이 프로를 검색/연결하는 플로우의 마찰 최소화
6. 리포트 가독성
AI 생성 리포트의 구조/레이아웃이 회원에게 유용한 정보를 전달하는가?
